## What Is a Chromebook Plus?

Google launched the "Chromebook Plus" designation in 2023 to set a quality baseline. Chromebook Plus models must include:

- Intel Core i3 or AMD Ryzen 3 (or higher)
- 8GB RAM (up to 4x faster than base Chromebooks)
- 128GB storage
- 1080p IPS display
- Built-in AI tools (Google Lens, Magic Eraser, Live Translate)

**Recommendation:** Get a Chromebook Plus if your budget allows. The performance gap between Plus and non-Plus is noticeable for multi-tab use.

## Chromebook vs. Windows Laptop: Which Should You Buy?

| Factor | Chromebook | Windows Laptop |
|--------|------------|---------------|
| Price | $199–$500 | $399–$1,500+ |
| Battery | 10–15 hours | 6–12 hours |
| Boot time | 6–10 seconds | 20–45 seconds |
| Software | Google apps + Android | Full Windows |
| Virus protection | Built-in | Needs antivirus |
| Offline use | Limited | Full offline |
| CS/Engineering apps | Not compatible | Compatible |

**Buy a Chromebook if:**
- Your school uses Google Classroom, Google Meet, and Google Docs
- You don't need Windows-only apps (Office365, Adobe CC, Visual Studio, etc.)
- Battery life and price are your top priorities

**Buy a Windows laptop if:**
- Your major requires specific software (AutoCAD, MATLAB, Visual Studio, Adobe Suite)
- You need offline functionality for coding, writing, or creative work
- You want to game

## Can Chromebooks Run Microsoft Office?

Yes — Microsoft Office 365 runs in the browser on any Chromebook. You get Word, Excel, PowerPoint, Teams, and Outlook through the web. If your school provides Microsoft 365 access, a Chromebook works seamlessly.

The only gap: Excel macros and complex VBA scripts don't run in the browser version. Finance and accounting students often need the desktop version.
