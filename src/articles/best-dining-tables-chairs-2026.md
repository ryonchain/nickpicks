## Dining Table Size Guide

Choose your table size based on room dimensions and seating needs:

| Table Size | Seats | Minimum Room Size |
|------------|-------|-------------------|
| 36–42" round | 2–4 | 9' x 9' |
| 48" round | 4–6 | 10' x 10' |
| 60" x 36" rectangular | 4 | 10' x 8' |
| 72" x 36" rectangular | 6 | 12' x 9' |
| 84" x 42" rectangular | 6–8 | 12' x 10' |
| Extendable (60–90") | 4–8 | 12' x 10' |

*Allow 36 inches between table edge and wall for chair pull-out clearance.*

## Frequently Asked Questions

**How big should a dining table be for a family of 4?**
A 60" x 30" rectangular table or a 48" round table seats 4 comfortably with room for serving dishes. For a family of 6, go to 72" x 36". For hosting flexibility, an extendable table that seats 4 normally and 6–8 extended is the best value.

**What type of dining table is easiest to clean?**
Glass tops wipe completely clean but show fingerprints and scratch easily. Laminate and MDF tops resist staining but can chip. Solid wood scratches but can be refinished. Performance lacquered finishes on solid wood are the best balance of durability and cleanability.

**How far from the wall should a dining table be?**
Allow 36" between the table edge and any wall or obstacle for comfortable chair pull-out. In tight dining rooms, 30" is the minimum.

**Are extendable dining tables worth it?**
Yes, for most households. The ability to seat more guests during holidays without maintaining a large table daily is practical. Modern extension mechanisms are reliable. Look for tables where the leaf stores underneath (built-in) rather than requiring separate storage.
