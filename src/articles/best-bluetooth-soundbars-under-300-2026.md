## Soundbar Buying Guide

**Room size matters:** 2.1 soundbars at 200–300W work well for rooms up to 300 sq ft. Larger rooms (400+ sq ft) benefit from 400W+ or a full 5.1 system with rear satellites.

**Connection priority:** Use HDMI ARC/eARC when possible — it provides better audio quality than optical and supports more formats. Optical is a reliable fallback for older TVs.

**Atmos vs Virtual Surround:** True Dolby Atmos requires Atmos-encoded content (Disney+, Netflix Atmos titles, 4K Blu-ray). Virtual surround simulates height effects from standard content. Neither requires ceiling speakers when done in a soundbar.
