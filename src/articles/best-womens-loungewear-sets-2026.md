## Loungewear Set Buying Guide

**Fabric options:**
- **French terry:** Lighter weight, slightly structured. Good for mild weather.
- **Fleece/sweatshirt fabric:** Warmest option. Best for fall/winter.
- **Velvet:** Softest feel. Best for indoor comfort only.
- **Waffle knit:** Textured, breathable. Casual but elevated look.
- **CozyChic/microfiber:** Ultra-soft luxury feel. More delicate.

**Fit styles:**
- **Oversized top + wide-leg pants:** Current trend. Comfortable and fashionable.
- **Fitted top + tapered jogger:** More athletic look. Works for gym transitions.
- **Relaxed top + relaxed pants:** Classic loungewear silhouette.

**Styling beyond the couch:**
- Matching lounge set + clean sneakers + tote bag = casual errand outfit
- Lounge set + trench coat = airport outfit
- Velvet lounge set + slides = elevated WFH look

**Care tips:**
- Fleece sets: wash cold, tumble dry low
- Velvet: gentle cycle, avoid high heat
- CozyChic: machine wash on delicate, lay flat to dry
