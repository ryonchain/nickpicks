## Dual Monitor Setup Configuration Guide

**Physical setup:**
1. Position primary monitor directly in front of you, at arm's length, screen top at eye level
2. Place secondary monitor at a slight angle (15–30°) to minimize head-turning
3. Match brightness and color temperature on both monitors (same warmth/kelvin setting)
4. Route cables through the arm or tie them together to keep the desk clean

**Windows 11 configuration:**
- Right-click desktop → Display settings → drag monitors to match physical layout
- Set both to the same scaling percentage (especially important for matching 4K monitors)
- Set primary display (Start menu, taskbar) to the monitor you face most directly

**macOS configuration:**
- System Settings → Displays → Arrangement → drag to match physical layout
- Set "Use as main display" on your primary monitor
- Consider enabling HiDPI mode on 4K displays under Display preferences

**Recommended accessory:** Dual monitor arm (Ergotron LX Dual or similar) — the most impactful addition to a dual monitor setup after the monitors themselves.
