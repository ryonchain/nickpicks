## Installation (No Plumber Required)

1. Unscrew the existing shower head (counter-clockwise; use pliers if stuck — protect chrome with a cloth)
2. Wrap new threads with 3 layers of Teflon tape (clockwise direction)
3. Screw on the new holder bracket by hand, then 1/4 turn with pliers
4. Attach the hose, then the shower head
5. Run water and check for drips at each connection

Total time: 5–10 minutes. No tools usually needed beyond a wrench.

## Best Hose Length for Shower Chairs

For users sitting in a shower chair, a 60-inch hose is the minimum useful length. 72–80 inches is ideal — it allows the user to reach the floor, rinse the shower, and clean with full freedom of motion from the seated position.
